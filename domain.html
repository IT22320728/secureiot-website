Literature Survey

Existing IoT security approaches often address only specific layers of protection such as static access control, gateway filtering, or isolated anomaly detection. This project explores integrating Zero Trust, SDN, trust scoring, and deception techniques into a unified framework.

Research Gap

Many existing systems are too complex or expensive for SOHO environments. There is a need for a lightweight, adaptive security solution that continuously verifies devices and responds dynamically to threats.

Research Problem

IoT environments are vulnerable due to weak authentication, poor monitoring, and lack of centralized control. Attackers can exploit these weaknesses for unauthorized access, data theft, and network disruption.

Research Objectives

  • Implement Zero Trust security for IoT networks
  • Ensure strong device authentication
  • Detect anomalies using behavioral analysis
  • Apply adaptive SDN-based policies
  • Use honeypots for threat intelligence

Methodology

  • Design layered IoT architecture
  • Use SDN controller (Ryu)
  • Collect traffic data
  • Analyze threats dynamically
  • Improve defense using honeypots
